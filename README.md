# Data Redundancy Removal System

## Overview
This project is developed as part of the Horizon TechX Cloud Computing Internship.

The system detects duplicate records before storing data and ensures that only unique and verified information is maintained in the database.

## Features
- Detects duplicate entries
- Prevents redundant data storage
- Stores only unique records
- Improves storage efficiency
- Uses SQLite database for record management

## Technologies Used
- Python
- SQLite
- Google Colab

## Project Objective
To reduce data redundancy and improve data accuracy by validating records before storing them in the database.
